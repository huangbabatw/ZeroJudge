### 題目

每組輸入共一行，內含有兩個整數 a, b，以空白隔開，a, b絕對值皆小於 106。

### 思考

1. 整數宣告 : int
2. 輸入指令 : cin , 同一行輸入多個東西可用 '>>' 串接
3. 輸出指令 : cout
4. 輸出時，可以直接放運算式，會印出運算結果。

### 範例

<pre><code>
#include &lt;iostream>
using namespace std;

int main() {
	int a,b;
	cin _________1___________;  
	
	cout << ___2___ << endl;

} 
</code></pre>

其中空格１要填輸入兩個變數的指令，空格２是兩數之和的運算式。
